It only works on Mac OS.

This is a Emacs key binding addon for Firefox 57+

Scroll

| keymap   | description      |
|----------+------------------|
| C-n or n | scroll down      |
| C-p or p | scroll up        |
| C-f      | scroll right     |
| C-b      | scroll left      |
| M-<      | scroll to top    |
| M->      | scroll to bottom |


Tab

| keymap  | description                   |
|---------+-------------------------------|
| M-f     | next tab                      |
| M-b     | previous tab                  |
| C-x C-f | new tab                       |
| C-x k   | close current tab             |
| t       | focus on the first input area |


Others

| keymap | description     |
|--------+-----------------|
| C-g    | focus on window |
| C-r    | refresh         |
| C-F    | history forward |
| C-B    | history back    |

These keybindings work in most webpages except pages such as 'about:xxxx'.
